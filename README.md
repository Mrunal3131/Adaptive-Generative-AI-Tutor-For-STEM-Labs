# 🎓 Adaptive Generative AI Tutor for STEM Labs

## 📌 Project Overview
The **Adaptive Generative AI Tutor for STEM Labs** is an intelligent virtual learning system that enables students to perform STEM experiments digitally with real-time AI guidance, adaptive hints, and automatic evaluation.

This system enhances conceptual understanding by combining **visual simulations** with **AI-powered tutoring**.

---

## 🚀 Features

- 🤖 Generative AI-based tutoring support  
- 🧪 Virtual STEM experiment simulations  
- 💡 Adaptive Hint System (step-by-step guidance)  
- 📊 Performance Evaluation & Feedback  
- 🔐 User Authentication (Login/Register)  
- 📈 Learning analytics based on student performance  

---

## 🛠️ Tech Stack

- Frontend: Streamlit  
- Backend:Python  
- Database:SQLite  
- AI Integration:Generative AI APIs  
- Other Tools:JSON, NumPy, etc.

## 🧠 How It Works

1. User logs into the system  
2. Selects an experiment  
3. Performs simulation steps  
4. AI Tutor provides guidance & hints  
5. System evaluates performance  
6. Feedback is generated
